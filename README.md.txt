# 📌 DSA Practice Reminder Bot (Telegram)

This is a no-code/low-code automation built with n8n to send a random DSA question to Telegram daily.

## 💡 What It Does
- Picks a random topic
- Picks a random question from that topic
- Sends it to my Telegram every morning

## 🔧 Tools Used
- Telegram Bot
- n8n
- Schedule trigger

## 🛠️ How to Use
1. Create a Telegram Bot with @BotFather
2. Set your Telegram chat ID and bot token
3. Clone this repo and import the workflow
